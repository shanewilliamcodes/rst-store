## Rules that make art embroider well (important)
- **Plain shapes, bold and simple.** No gradients, no shadows, no 3D, no photo
  realism, no thin hairlines. Embroidery is thread — think bold sticker, not
  fine illustration.
- **Transparent or plain white background.** One design per file.
- **Square image, centered, with margin around it.** At least 1500×1500 px.
- **Color doesn't have to be perfect** — the shape is what matters; I set the
  actual thread color. Bright/flat color (or even solid black) is fine.

## Easiest ways to make them

**Option A — AI image tool (free, ~10 min).** I recommend **Recraft**
(recraft.ai — it does flat vector art with transparent backgrounds and PNG/SVG
export) or **Ideogram** (ideogram.ai). Paste these prompts, one per image:

- **Heart:** `Flat vector icon of a simple elegant heart, solid bright pink fill, smooth rounded classic shape, no gradient, no shadow, no text, centered, transparent background, minimalist, bold and clean for embroidery`
- **Mustache:** `Flat vector icon of a classic curled gentleman's mustache silhouette, solid orange fill, symmetrical, smooth bold shape, no face, no text, centered, transparent background, minimalist, clean for embroidery`
- **Baby bottle:** `Flat vector icon of a cute simple baby bottle, bold clean outline, bright blue, with a nipple top and a few measurement lines, symmetrical, no text, centered, transparent background, minimalist, clean for embroidery`
- **ABC blocks:** `Flat vector icon of three toy alphabet blocks reading A B C, playfully stacked, each a solid bright color (lime green, yellow, pink) with dark letters, bold simple shapes, no gradient, transparent background, minimalist, clean for embroidery`

Generate a few, pick the best, download as PNG (transparent background).

**Option B — Fiverr (hands-off, ~$10–25).** Search "vector logo / icon design"
or "embroidery digitizing." Send them this brief and the 4 descriptions above.
You'll get polished files.
