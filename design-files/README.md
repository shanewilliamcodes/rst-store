## Notes for embroidery
- **Size:** left-chest signature ~2.5–3 in wide; sleeve "RST" ~1.5–2 in wide.
- **Neon thread:** Printful matches to the closest Madeira thread. The pink,
  orange, and aqua have good fluorescent matches; if a shade can't go full neon,
  pick the brightest available — we can tune the on-site mockup to match.
- **Smallest details:** the bottle outline and the A/B/C letters are the finest
  detail. If Printful's review flags them as too small to stitch cleanly, bump
  the chest size up to 3 in, or I can thicken those shapes — just tell me.
- **Sleeve "RST":** easiest path is Printful's built-in embroidery *text* tool
  (type "RST", serif font) instead of uploading the file — either works.
